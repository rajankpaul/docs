# Flask

> microservice webserver

## Virtal Environments

### venv (Python 3)
```
sudo apt-get install python3-venv
mkdir myproject
cd myproject
python3 -m venv venv
. venv/bin/activate
```

### virtualenv (Python 2)

```
sudo apt-get install python-virtualenv
python2 -m virtualenv venv
. venv/bin/activate
```

## Installation

```
sudo pip3 install flask
```
