# pyenv

## Installation

```
curl -L https://pyenv.run | bash
```

## Setup

```
$ echo 'export PATH="~/.pyenv/bin:$PATH"' >> ~/.bashrc
$ echo 'eval "$(pyenv init -)"' >> ~/.bashrc
$ echo 'eval "$(pyenv virtualenv-init -)"' >> ~/.bashrc
$ source ~/.bashrc
```

## Install Python Version

```
pyenv install 3.6.2
```
